Spring의 핵심 개념인 의존성 주입(Dependency Injection)을 학습합니다. XML 기반 설정과 애너테이션 기반 설정(@Autowired)을 각각 사용하여 객체 간 의존 관계를 주입하고, 그 차이점을 이해합니다.

\[해야 할 일\]

- XML(applicationContext.xml)로 빈을 정의하고, 서로 다른 객체 간 의존성을 연결합니다.
- 동일한 코드를 애너테이션(@Autowired) 기반으로 변경합니다.
- 두 방식의 차이와 장단점을 비교합니다.

\[산출물\]

- XML 설정 파일 및 Java 코드
- 실행 결과 요약 (예: “XML 기반 DI와 애너테이션 기반 DI 모두 정상 동작”)
